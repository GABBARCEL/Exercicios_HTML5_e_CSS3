HTML 5 semântico:

No HTML 5, as tags se tornaram semânticas.


Isso significa que o HTML deve tratar apenas do conteúdo e seu significado, e não do formato do conteúdo

A partir do HTML 5, quem cuida do formato do conteúdo é o CSS, com a tag <style>

tags antigas do HTML 4, como <u>, ficaram obsoletas. Elas até podem funcionar, mas não tem semântica, só fomato. Portanto, é errado usalas
