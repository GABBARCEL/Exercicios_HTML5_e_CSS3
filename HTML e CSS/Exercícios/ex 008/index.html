Principais formatações

Negrito / destaque:

Nesta frase, temos um termo em negrito, usando a tag <b> (Não semântico / sem sentido)

Nesta frase, temos um termo em negrito, usando a tag <strong> (semântico / com sentido)

Itálico / Ênfase:

Nesta frase, temos um termo em itálico, usando a tag <i> (não semântico / sem sentido)

Nesta frase, temos um termo em itálico, usando a tag <em> (semântico / com sentido)

Sublinhado:

Nesta frase, temos um termo sublinhado, usando a tag <u> (não semântico / sem sentido)

Nesta frase, temos um termo sublinhado, usando a tag <> (semântico / com sentido)

Marcação:

Podemos também marcar partes de textos com a tag <mark> (semântico)

É possível mudar a cor ta marcação, alterando a tag com outra tag, a <style>. Quando usamos essa tag, estamos trabalhando com CSS

Texto Grande e Pequeno

É possível deixar um texto grande com a tag <big> (Não semântica / sem sentido)

Da mesma forma, é possível deixar um texto pequeno com a tag <small> (ainda semântico)

Texto riscado / deletado

Nesta frase, temos um texto riscado, usando a tag <del> (semântico)

Texto inserido

Nesta frase, temos um texto inserido, usando a tag <ins> (semântico)

Texto subescrito e texto superior

texto superior:

pode-se usar a tag <sup> para elevar o texto, formando expressões matemáticas e físicas: exemplo x2


Texto subescrito:

pode usar a tag <sub> para deixar o texto abaixo, ou subescrito. Exemplo: H2O
